Hi, welcome to our Sermo Chat app.
To run the app, run:
1. Sermo
2.SermoApp

the "Sermo" will open the API and upload the server.
the "SermoApp" will start the app, allowing to go to the ratings page and to the chat app login.

please make sure to run the Sermo folder before launching the SermoApp, otherwise the login will not work and the ratings will not be saved.
